# Philosophy Club Whiskey Management System

## Overview

This project aims to manage the Philosophy Club's whiskey inventory and member contributions. It features a modern frontend built with HTML, CSS, and JavaScript, styled using Bootstrap. The frontend allows users to:

- View a list of stored bottles
- View a list of club members
- Add new bottles and their prices
- Add new members and their statuses
